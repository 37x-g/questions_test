【名称寓意】
青灯为伴，黄卷为友，孜孜不倦，学以致用。

当前版本：v1.0.0

【软件简介】
本软件是一款专为期末复习、知识竞赛等设计的复习做题工具。
支持从TXT/Word/JSON导入题库，提供随机抽题、收藏斩杀、搜索题目、
修改答案、导出错题等功能。

【主要功能】
--------------------------------------------------------------------------------
📂 导入题库     - 支持 TXT、Word、JSON 格式，导入后自动保存
🎲 随机抽题     - 从题库中随机抽取题目
⏪ 上一题       - 返回上一道做过的题目
📖 显示答案     - 点击按钮显示参考答案
✏️ 输入答案     - 在输入框中输入你的答案
✅ 提交答案     - 提交后自动判断对错并显示参考答案
⭐ 收藏本题     - 将题目加入收藏本，方便复习错题
⚔️ 斩杀本题     - 斩杀后题目不再被随机抽到
📋 收藏本       - 查看、做题、移除收藏题目
🔍 搜索题目     - 按关键词搜索题目
📎 导出错题     - 将收藏本导出为 Excel 文件
✏️ 修改答案     - 修改题目答案，自动保存
🔄 重置进度     - 清除得分、收藏、斩杀记录
🤖 AI助手      - 跳转DeepSeek，复制问题提问
🔥 每日打卡     - 自动记录连续学习天数
🔔 提示开关     - 可开启/关闭各类提示弹窗
🎨 主题切换     - 切换正常/护眼/黑夜模式
⚙️ 快捷键设置   - 自定义键盘快捷键
🔄 自动更新     - 检测GitHub新版本

【快捷键】
--------------------------------------------------------------------------------
空格 / 回车    - 下一题
Ctrl+Q        - 上一题
S             - 显示答案
A             - 提交答案
F             - 收藏/取消收藏
D             - 斩杀本题
Esc           - 退出程序
Ctrl+F        - 搜索题目
Ctrl+E        - 导出收藏本

可在"⚙️ 快捷键设置"中自定义

【题库格式】
--------------------------------------------------------------------------------
[题目1]
问题：什么是动校正？
答案：将反射波传播时间校正到自激自收时间的处理过程

[题目2]
问题：下列哪种波传播速度最快？
选项：A.纵波 B.横波 C.面波 D.声波
答案：A

【文件说明】

saved_questions.json    - 存储导入的题库

quiz_stats.json         - 存储统计信息
quiz_settings.json      - 存储软件设置
